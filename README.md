# kryptorious-devkit (npm)

> Project scaffolding CLI for Node.js and TypeScript.

[![npm](https://img.shields.io/npm/v/kryptorious-devkit)](https://www.npmjs.com/package/kryptorious-devkit) [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Part of the [Kryptorious developer toolkit](https://kryptorious.gumroad.com/l/jbvet) — 31 open-source tools, one $9 lifetime license.

## Install

```bash
npm i -g kryptorious-devkit
```

## Quickstart

```bash
devkit init my-app --template node
# -> scaffolds a Node/TypeScript project with lint, test, and CI pre-configured
```

## Commands

| Command | Description |
|---------|-------------|
| `devkit init <name> --template node` | Scaffold a new project. |
| `devkit init <name> --template ts` | Scaffold a TypeScript project. |
